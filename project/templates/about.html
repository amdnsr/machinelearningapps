This website provides a User Interface to run the following Machine Learning Applications

Caption Generation

This service generates captions for images using a deep neural network model trained in tensorflow.

Cartoonization

This service cartoonizes images using the cartoonizer developed by Filip Andersson, Jönköping University.
The Generative Adversial Network (GAN) model was trained on over 60,000 images from works by Hayao Miyazaki at Studio Ghibli.

Text Summarization

This service summarizes a given text using the deep learning model called Bart by Facebook
